# path-dependence
Path-Dependent Strain Prediction in Shape Memory Alloys.

This repository contains MATLAB code and experimental datasets used to model
path-dependent strain behavior in shape memory alloys. The framework
predicts the strain evolution along thermomechanical paths using experimental data 
obtained from tensile uniaxial constant force thermal cycling (UCFTC) experiments, also referred to
as iso-stress heating and cooling (ISHC) or isobaric heating and cooling (IBHC) tests. If available,
detwinning experiments are also used to characterize martensite reorientation/detwinning behavior. In
these experiments, the material is first cooled into martensite at a constant minimum stress (5 MPa),
followed by reorientation/detwinning of martensite up to the reversible stress limit before reversing 
the loading path. To demonstrate the modeling framework, experimental datasets for Ni50.9Ti49.1 are
provided, along with experimental thermomechanical paths used to investigate path-dependent behavior
in this alloy.

The code was developed to support research on elastocaloric refrigeration and
thermodynamic modeling of shape memory alloys.

## Running the Model

Details of each script/function are provided in the summary at the top 
of each file to aid the user.

The strain prediction framework is executed in two stages.

### 1. Characterize material properties

Run the characterization scripts to construct phase boundaries and material
properties from experimental data. The ACHANGE_characterize script is intended to
be edited interactively based on the visual outputs generated by ARUN_characterize.
You must save the ACHANGE_characterize script after interactive edits before continuing the ARUN script.

```matlab
ACHANGE_characterize
ARUN_characterize
```

This step generates the phase boundary, material property data, and initial strain
surfaces used in the strain prediction framework.

---

### 2. Run strain prediction

Execute the prediction script after specifying the (T, σ) "corner" points of the 
paths of interest:

```matlab
ARUN_predict
```

This script performs the following tasks:

1. Construct thermomechanical loading paths  
2. Populate phase fractions along each path  
3. Reconstruct strain surfaces  
4. Apply path-dependent strain corrections  
5. Generate predicted strain histories  

---

## Output

Results are stored within the `Paths` structure and include:

- Predicted strain evolution along each thermomechanical path
- Phase fraction histories
- Additional path and material property data used in the prediction workflow
